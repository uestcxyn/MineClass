MineClass
=========

MineClass，直译就是“我的班级”，名称灵感来自MC游戏，项目灵感来自现在班级通知不到位，学生无法快速高效获取

通知。项目要解决的主要问题是班级通知，在此基础上建设班级的一个圈子，记录班级的点点滴滴。

##提交说明
 - /docs----此目录下放API文档
 - 每个人一个分支(branch)提交，比如我是master分支
 - 提交描述我推荐如下格式
>第一行是对改动的简要总结，建议长度不超过 50，用语采用命令式而非过去式。
第二行空行。
第三行开始，是对改动的详细介绍，可以是多行内容，建议每行长度不超过 72。
最后签个名什么的
